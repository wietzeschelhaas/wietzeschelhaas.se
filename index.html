Web server from scratch in python

This website is hosted on a Raspberry Pi 3 b+ running a web server made in python using only the low-level socket library. It is multithreaded and supports the most common media types such as images and videos, though it has some trouble serving larger files. As of right now the webserver only supports GET request, attempting to send a HTTP request using a method other than GET will result in the server generating a response with the error 501 Not Implemented set in the header.
