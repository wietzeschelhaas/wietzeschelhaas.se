Web Server in Python from Scratch

This website is hosted on a Raspberry Pi 3 b+ that is running a web server created in Python by using only the low-level socket library. It is multithreaded and supports the most common media types such as images and videos. However, it has some trouble with serving larger files. As of right now, the web server only supports GET request. In other words, attempting to send a HTTP request using a method other than GET will result in the server generating a response with the error "501 Not Implemented" set in the header.

Neural Evolution of Flappy Bird

I made the game "Flappy Bird" in LibGDX and then applied the concept of neural evolution to create a bird that beats the game. A simple neural network was developed from scratch and used to determine if a bird should jump or not. The input to this network is the x,y position of the bird, and the x,y position of the pipes. 50 birds are spawned in one generation while each of them has its own neural network. When a bird dies, it gets assigned a fitness value that is simply the amount of time the bird managed to survive. This fitness value is used to determine how likely the bird's genetic code (its neural network) is passed on to future generations. When all birds are gone, a new generation is spawned. The birds in the new generation are created by randomly selecting two parent birds from the previous generation and combining their neural networks. The neural networks that produced higher fitness values are more likely to be selected as parents. The newly created birds are slightly mutated, and a new generation of birds is confronted with the challenges the game "Flappy Bird" poses. Given enough generations, the algorithm produces a neural network that is able to beat the game.
